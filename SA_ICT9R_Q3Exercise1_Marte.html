HTML

HyperText Markup Language

HTML is also known as HyperText Markup Language. It lets us create web pages. HTML instructs the web browser on how to display your web pages.

CSS

Cascading Style Sheets

CSS can be internal, inline, or external. Innernal CSS is found in the head tag. Inline CSS is found in the line you want to put the CSS. External CSS is a separate file that uses .css extension.

Bootstrap

Bootstrap Version 4

Bootstrap makes it easier to develop websites or web pages. It lets us create responsive designs. Bootstrap is released on August 19, 2011 and is developed by Mark Otto and "fat".
